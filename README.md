Problem Statement

This is a regression problem where the goal is to predict a numerical writing score using a mix of:

Categorical features (gender, race/ethnicity, lunch type, etc.)

Numerical features (math score, reading score)

The model is currently trained only to predict writing score based on the provided inputs.

Features

Predicts student writing score using multiple input factors

Trains and evaluates multiple regression models

Automatically selects the best-performing model

Clean and simple Flask web interface

Supports custom user input through a form-based UI
